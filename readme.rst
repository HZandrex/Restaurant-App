Wat is de Web App?
###############################
Deze webapplicatie is geschreven in Java code gebaseerd op het MVC concept.
Het thema van de applicatie is restaurant.
Enkele GRASP designs worden toegepast zoals creator-pattern, expert-pattern, ...
De basis functies van deze app zijn: test gegevens genereren, lijst van personeelsleden tonen,
zoeken in bestellingen (op datum en tafel), toon bestel details, nieuwe bestellingen aanmaken, ...
